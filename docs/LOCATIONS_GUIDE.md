# Довідник водойм Fishoria

Цей довідник допоможе вам обрати найкращу водойму для риболовлі, дізнатися про переважаючі види риб, глибини та специфічні умови на кожній локації.

---

## 1. Лісове озеро (Forest Lake)

- **Опис:** Затишне природне озеро, оточене стародавніми деревами. Характеризується нерівним дном, багатою рослинністю та різноманітною фауною. Перша і найпростіша водойма для початку гри.
- **Діапазон глибин:** 0.5 м — 2.7 м
- **Особливості умов:**
  - Вода має середню прозорість (видимість 65%).
- **Шанс витягнути сміття:** 8%
- **Щільність риби:** Середня (максимум до 80 особин одночасно).

### Популяція риб:

| Вид риби             | Орієнтовна частка (%) | Оптимальна глибина лову |
| :------------------- | :-------------------: | :---------------------: |
| **Карась (Crucian)** |          40%          |      0.5 м - 2.7 м      |
| **Плітка (Roach)**   |          30%          |      0.5 м - 1.8 м      |
| **Окунь (Perch)**    |          15%          |      0.5 м - 2.2 м      |
| **Щука (Pike)**      |          10%          |      0.5 м - 2.7 м      |
| **Йорж (Ruffe)**     |          5%           |      1.0 м - 2.7 м      |

---

## 2. Рибгосп (Fish Farm Reservoir)

- **Опис:** Штучне водосховище, створене та кероване спеціально для розведення риби. Має відносно рівне дно, дуже чисту воду та високу щільність рибного населення (особливо великих видів).
- **Діапазон глибин:** 0.3 м — 4.0 м
- **Особливості умов:**
  - Вода надзвичайно прозора (видимість 90%), тому риба краще бачить наживку.
- **Шанс витягнути сміття:** Лише 2%
- **Щільність риби:** Висока (максимум до 110 особин одночасно).
